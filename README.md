# dishdash1
A full-stack cooking timer game where players race against 
the clock to complete recipe steps.

## Tech stack 
- Frontend: HTML, CSS , JavaScript
- Backend: Node.js, Express
- Database: PostgreSQL(Supabase)
- Deploy: Vercel (frontend), Render (backend)

## Features
- 6 cuisines 
- Per-step countdown timers
- Score system with bonuses
- Win/lose result screen
- Fully responsive
